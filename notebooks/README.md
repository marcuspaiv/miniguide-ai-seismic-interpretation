# 🪨 Reservoir Image Analysis with Python

This project presents a study on reservoir characterization using image processing techniques applied to thin section images.

## 📌 Objectives

* Analyze pore distribution using image histograms
* Estimate porosity based on pixel intensity
* Segment grains and pores using thresholding and watershed algorithms
* Evaluate representative elementary volume (REV)

## 🧠 Methods

* Histogram analysis (RGB and grayscale)
* Image thresholding
* Porosity estimation from pixel classification
* Watershed segmentation for grain detection

## 🛠️ Technologies

* Python
* NumPy
* Matplotlib
* Scikit-image

## 📊 Results

The analysis allowed:

* Estimation of porosity, grains, and cement proportions
* Visualization of pore distribution
* Segmentation of grains using watershed algorithm

## 🚀 Insights

Watershed-based segmentation provided more accurate results compared to simple thresholding, especially for complex grain distributions.

## 📂 Dataset

Thin section image provided during the course.
